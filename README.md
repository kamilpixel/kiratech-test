# Kiratech App Test

## Online preview


## Requirement to run in local
Please install `node` and `npm` in local
- node v18.13.0
- npm v8.19.3

## Setup in local

```bash
# set env in local
copy `.env.example` and rename to `.env`

# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev
```
